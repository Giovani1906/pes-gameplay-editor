# pes-gameplay-editor
An editor that modifies the values of the binary files located in the "dt18" CPK file of the Pro Evolution Soccer games.

## Requirements
* Python 3.10+
* [PySide6 6.7.2+](https://doc.qt.io/qtforpython-6/)
